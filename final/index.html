Lights Out

Click a tile to toggle it and its neighbors. Turn all the lights off to win!

Moves: 0
New puzzle! Turn all the lights off.

How to Play

  1. Click any tile on the board.
  2. The clicked tile and its up, down, left, and right neighbors will toggle.
  3. Yellow tiles are ON, dark tiles are OFF.
  4. Your goal is to make all tiles turn OFF.
  5. When every tile is off, a pop-up will say "You win!".

Hint – Random but Solvable Board

The puzzle does not start with a completely random pattern, because some random boards in Lights Out are impossible to solve. Instead, the game begins from an all-off board and the program simulates a series of random valid clicks using the same rules as the player. Any board created this way can always be solved, so each new game is both random and solvable.

Font Notes

To satisfy the “font craziness” requirement, this page uses Google Fonts, including Press Start 2P for the retro title and Orbitron for the rest of the text. The fonts are loaded using an @import statement at the top of styles.css:

@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&family=Orbitron:wght@500&display=swap');

When the page loads, the browser downloads these fonts from Google’s servers and applies them to the elements styled in the CSS rules.

Project Addendum

This project implements a playable Lights Out game using HTML, CSS, and JavaScript. The board is generated dynamically, and each tile toggles itself and its neighbors when clicked. To ensure the game is solvable, I began with an all-off board and applied random simulated clicks, which guarantees a valid configuration.

The game shows the required "You win!" alert when all lights are off. I added small embellishments such as a move counter and a status message to make the gameplay feel more interactive. I also used Google Fonts to satisfy the font requirement and included the footer script to display the last modified time.

This project helped me practice DOM manipulation, event handling, styling, and deploying a webpage through GitHub Pages.
