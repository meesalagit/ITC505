HOW TO PLAY

Objective: Turn all the lights off. A light is considered “off” when the square is completely dark.

Basic play: Clicking a square toggles that square and its up, down, left, and right neighbors. Yellow tiles are ON, dark tiles are OFF. When every tile is dark, a pop-up will say “You win!”.

Hint: The board you see is not purely random. The game starts from an all-off board and then applies random valid moves using the same rules as the player. Because these are legal moves, the final pattern is always solvable.

Font notes: This page uses the retro arcade font Press Start 2P for the title and a clean sans-serif font for the body text. Both fonts are loaded from Google Fonts using a CSS @import rule at the top of styles.css, which creates the intentional “font craziness” required by the assignment without showing the full URL to the user.

Project Addendum

This project implements a playable Lights Out game using HTML, CSS, and JavaScript. The board is generated dynamically, and each tile toggles itself and its neighbors when clicked. To ensure the game is solvable, I began with an all-off board and applied random simulated clicks, which guarantees a valid configuration created by real game moves.

The game shows the required "You win!" alert when all lights are off. I added small embellishments such as a move counter and a status message to make the gameplay feel more interactive. I also used Google Fonts to satisfy the font requirement and included the footer script to display the last modified time of the page.

This project helped me practice DOM manipulation, event handling, styling with CSS, and deploying a webpage through GitHub Pages while debugging issues like caching and file-path problems.
